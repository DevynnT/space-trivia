# Space Trivia Game

Crudely made game using pure JavaScript. For IDSP-2.

## How to Run

```bash
git clone https://github.com/DevynnT/space-trivia
```

In the directory:

```bash
npm install
```

### Run Game

```bash
npm start
```

Runs on `localhost:8000` by default. Create a .env file with `PORT` to chnage the port.
